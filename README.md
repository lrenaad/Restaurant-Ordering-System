# Restaurant Ordering & Delivery System

A desktop application developed in **Java** that simulates real restaurant operations, including menu management, customer ordering, and delivery processing using fundamental data structures.

---

## 🛠️ Data Structures Implemented

* **Singly Linked List**
  * Used to store and manage menu and order items dynamically.
  * Supports searching for orders, calculating total costs, removing items, and displaying lists.
* **Queue (FIFO)**
  * Used to manage the delivery process.
  * Ensures that orders are processed and delivered in the exact sequence they were received.

---

## 🚀 Key Features

* **Menu Exploration:** Browse available restaurants and view their menus.
* **Order Creation:** Create new customer orders by selecting items and adding customer details.
* **Delivery Management:** Add pending orders to the delivery queue and deliver the next order in line.
* **Order Search & Control:** Search for specific orders instantly using a unique Order ID.
* **Data Management:** Remove menu items from a restaurant or cancel specific orders from the delivery queue.
* **Queue Visualization:** Display all orders currently waiting in the delivery line.
* **Financial Insights:** Automatically calculate and display the total revenue generated from all processed orders.
